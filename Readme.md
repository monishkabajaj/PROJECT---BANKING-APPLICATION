## Project Title
**Banking Application for Account Management**

---

## Problem Statement
Design and implement a banking application that allows users to manage their bank accounts through operations such as:
- Creating new accounts  
- Depositing money  
- Withdrawing money  
- Viewing account details  
- Updating contact details  

---

## Features
- Unique account numbers (start from 1001).  
- Input validation for deposit/withdraw.  
- Menu-driven console interface.  
- Contact details update (email, phone).
